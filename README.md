# CRUD Visitantes

### Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/ngiovana/visitors-crud.git
   ```

2. Navegue ao diretório backend
   ```bash
   cd backend
   ```
   
4. Inicie o projeto
   ```bash
   dotnet run
   ```

5. Em outro terminal, navegue ao diretório frontend
   ```bash
   cd frontend
   ```

6. Instale as dependências
   ```bash
   npm install
   ```

7. Inicie o projeto
   ```bash
   npm run start
   ```
